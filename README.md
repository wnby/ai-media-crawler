#  AI Media Crawler (基于 Crawl4AI 的爬虫)

这是一个基于开源库 [Crawl4AI](https://github.com/unclecode/crawl4ai) 构建的异步网页爬虫工具。专门用于抓取中文 AI 公众号（如**量子位**、**新智元**）的文章资讯。


##  支持媒体源

当前代码内置支持/兼容以下站点的解析逻辑：
- ✅ **量子位** (QbitAI)
- ✅ **新智元** (XinZhiYuan / Aiera)
